# MISSION : COPIER LE DESIGN EXACT DE L'ANCIEN SITE

## CONTEXTE
Le site CannaZen a été reconstruit en Vite+React mais le design est COMPLÈTEMENT DIFFÉRENT de l'original. L'ancien site (build HTML/CSS compilé) est ENCORE dans le projet sous forme d'archives. Tu dois EXTRAIRE l'ancien design et le REPRODUIRE à l'identique.

## ÉTAPE 1 — RETROUVE L'ANCIEN SITE

Cherche dans le projet les archives et fichiers de l'ancien site :
```bash
ls -la *.zip *.tar.gz
ls -la cannazen-hostinger.zip cannazen-vercel.tar.gz cannazen-vercel/ CANNAZEN_FINAL.tar.gz
```

Extrais la plus complète (probablement cannazen-hostinger.zip ou CANNAZEN_FINAL.tar.gz) dans un dossier temporaire :
```bash
mkdir /tmp/old-site
# Si zip :
unzip cannazen-hostinger.zip -d /tmp/old-site
# Si tar.gz :
tar xzf CANNAZEN_FINAL.tar.gz -C /tmp/old-site
```

## ÉTAPE 2 — ÉTUDIE CHAQUE DÉTAIL DU DESIGN ORIGINAL

Ouvre et lis ATTENTIVEMENT :
1. Le fichier index.html principal — note le HTML exact du header, hero, sections, footer
2. TOUS les fichiers CSS — note CHAQUE couleur, font, taille, espacement, bordure, ombre
3. Les images et le logo — copie-les dans public/
4. Les fonts utilisées (Google Fonts ou locales)

Note PRÉCISÉMENT :
- La couleur exacte du fond (hex)
- La couleur exacte du header (hex)  
- La couleur exacte des boutons (hex)
- La couleur exacte du texte (hex)
- Les fonts exactes (nom, poids, taille)
- Le style exact du logo (image ? SVG ? texte stylisé ?)
- Le bandeau défilant exact (texte, couleurs, vitesse)
- Le style exact des cards produits
- Le style exact de la page connexion
- Le style exact du footer

## ÉTAPE 3 — COPIE LE LOGO ET LES ASSETS

```bash
# Copie TOUS les assets de l'ancien site
cp -r /tmp/old-site/images/* public/images/ 2>/dev/null
cp -r /tmp/old-site/assets/* public/assets/ 2>/dev/null
cp -r /tmp/old-site/public/* public/ 2>/dev/null
# Cherche le logo partout
find /tmp/old-site -name "logo*" -o -name "*.svg" -o -name "favicon*" | head -20
```

Si le logo est une image (PNG/SVG), copie-le tel quel dans public/ et utilise-le dans le header. NE CRÉE PAS un nouveau logo. Utilise L'ORIGINAL.

## ÉTAPE 4 — APPLIQUE LE DESIGN EXACT

Maintenant, modifie le site Vite+React actuel pour reproduire PIXEL PAR PIXEL le design original :

### Header

### Page d'accueil

### Cards produits
