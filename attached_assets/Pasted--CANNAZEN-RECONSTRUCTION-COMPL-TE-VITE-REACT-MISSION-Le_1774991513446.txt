# 🔥 CANNAZEN — RECONSTRUCTION COMPLÈTE VITE + REACT

## MISSION
Le projet actuel est un build figé (HTML/CSS/JS compilé + Ecwid). Il faut le RECRÉER from scratch en Vite + React + TypeScript. Le design existant DOIT être préservé : dark theme, accents verts, CannaZen branding. Ecwid est SUPPRIMÉ et remplacé par un catalogue interne.

## ÉTAPE 0 — CRÉER LE PROJET VITE + REACT

Crée un nouveau projet Vite + React + TypeScript :
```bash
npm create vite@latest . -- --template react-ts
npm install
npm install react-router-dom firebase tailwindcss @tailwindcss/vite lucide-react
```

Configure Tailwind dans vite.config.ts :
```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})
```

Dans le fichier CSS principal, ajoute :
```css
@import "tailwindcss";
```

## DESIGN SYSTEM

Couleurs :
- Background principal : #0a0a0a (noir profond)
- Cards/sections : #111111
- Bordures : #222222
- Texte principal : #ffffff
- Texte secondaire : #888888
- Accent principal : #22c55e (vert, pour les boutons principaux, comme vu sur le site actuel)
- Accent hover : #16a34a
- Accent secondaire : #9333EA (violet, pour badges et détails)
- Erreurs : #ef4444
- Warnings : #f59e0b
- Success : #22c55e

Fonts :
- Titres : system-ui ou "DM Serif Display" (si disponible via Google Fonts)
- Corps : system-ui ou "Plus Jakarta Sans"

Bordure radius : 12px pour les cards, 8px pour les inputs/boutons

## FIREBASE AUTH — lib/firebase.ts

Configuration ROBUSTE : le site ne crashe JAMAIS si Firebase n'est pas configuré.

```typescript
import { initializeApp, getApps, FirebaseApp } from 'firebase/app';
import { getAuth, GoogleAuthProvider, Auth } from 'firebase/auth';
import { getFirestore, Firestore } from 'firebase/firestore';

let app: FirebaseApp | null = null;
let auth: Auth | null = null;
let db: Firestore | null = null;
let googleProvider: GoogleAuthProvider | null = null;

const firebaseConfig = {
  apiKey: import.meta.env.VITE_FIREBASE_API_KEY,
  authDomain: import.meta.env.VITE_FIREBASE_AUTH_DOMAIN,
  projectId: import.meta.env.VITE_FIREBASE_PROJECT_ID,
  storageBucket: import.meta.env.VITE_FIREBASE_STORAGE_BUCKET,
  messagingSenderId: import.meta.env.VITE_FIREBASE_MESSAGING_SENDER_ID,
  appId: import.meta.env.VITE_FIREBASE_APP_ID,
};

export const isFirebaseConfigured = Boolean(firebaseConfig.apiKey && firebaseConfig.projectId);

if (isFirebaseConfigured && getApps().length === 0) {
  try {
    app = initializeApp(firebaseConfig);
    auth = getAuth(app);
    db = getFirestore(app);
    googleProvider = new GoogleAuthProvider();
    googleProvider.setCustomParameters({ prompt: 'select_account' });
  } catch (e) {
    console.warn('Firebase init failed:', e);
  }
}

export { app, auth, db, googleProvider };
```
